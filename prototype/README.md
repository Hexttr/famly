# Famly Prototype

Интерактивный UX-прототип приложения Famly (React + Vite).

## Запуск

```bash
cd prototype
npm install
npm run dev
```

Откройте http://localhost:5173 — мобильный вид 375px.

## Экраны

- **Free (12):** Onboarding, Home, Quick Add, Operations, Operation Detail, Budget, Category Budget, Categories, Accounts, Reports, Settings, Backup/Export
- **Premium (6):** Paywall, Family Hub, Family Member, Split Expense, Balances IOU, Advanced Reports

## Design System

Токены в `src/theme.ts` — контракт для Jetpack Compose.
