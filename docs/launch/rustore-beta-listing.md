## Full description

Famly — приложение для семейного бюджета и учёта расходов.

**Beta:** семья, синхронизация, split расходов, IOU, rollover бюджета, расширенная аналитика и экспорт — **бесплатно**.

- Safe-to-spend и бюджет по категориям
- Быстрый ввод операций и виджет
- Семья до 6 человек с облачной синхронизацией
- Split расходов и учёт долгов
- JSON / CSV / Excel экспорт
